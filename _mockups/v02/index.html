FAQ

Do I need to be “good with computers”?

No. I teach at the pace you need. The goal is confidence and repeatable steps, not speed.

What do I need for the class?

A computer with internet, and Zoom (or Teams). If you prefer, we can also work from an iPad.

What will I learn first?

Basics: how to ask for help clearly, how to check results, and how to use AI for real tasks like writing, planning, and questions.

Will you help me set up a simple workflow?

Yes. I’ll help you create a small, repeatable routine you can keep using after the session.
